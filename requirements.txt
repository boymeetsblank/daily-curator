# requirements.txt — List of Python packages this project needs
#
# HOW TO INSTALL:
#   Run this command in your terminal (inside the daily-curator folder):
#   pip install -r requirements.txt
#
# WHAT THIS MEANS:
#   "pip" is Python's package manager — like an app store for Python libraries.
#   The -r flag means "install everything listed in this requirements file".
#   Each line below is one package to install.
#
# ─────────────────────────────────────────────────────────────────────────────
