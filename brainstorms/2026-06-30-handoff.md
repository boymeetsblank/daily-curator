## What Blank is (unchanged)
Consumer AI-editor news reader (PWA). The AI **selects + organizes** (ranks, clusters,
de-dups) — never authors; **real headlines only**. Engine (`run_pipeline.py`, every 10 min):
ingest → trends → OG-enrich → triage (Haiku KILL/ESCALATE) → score (Sonnet 1–10 + topic
tags) → tag-backfill → moment. State in `blank.db` (committed each run); feed built by
`deploy-pages.yml` → `picks_data.json` → `index.html`. **Load-bearing rule: global-score-once
— never per-user LLM scoring.**

## Working mechanics
- `git pull --rebase origin main` first (engine commits `blank.db` ~every 10 min; `git stash`
  a dirty `blank.db`). Commit on `main`; **never commit `.env`**; keep `CHANGELOG.md` ≤ 20
  entries (add newest at top).
- **PowerShell can't do heredocs / trips on quotes+parens** — write multi-line commit messages
  to a file and `git commit -F <file>`.
- Verify engine changes on a **copy** of `blank.db`, never the authoritative file.
- Tunable single constants: `DECAY_HOURS_PER_POINT` + `PER_SOURCE_CAP` (`db.py`),
  `OG_MAX_ATTEMPTS` / `OG_ENRICH_PER_RUN` / `TREND_REFRESH_MINUTES` (`ingest.py`),
  `NICHE_CAP` (`sections.py`), `MOMENT_*` (`moment.py`).
- **Notion:** keep the **Session Log** (dated entries) + **Roadmap** DB updated every session,
  unprompted — it's the live source of truth. Connector = `mcp__claude_ai_Notion__*`; verify
  with `fetch` id `self`. Blank parent page:
